# UI Design Guide Skill

前端 UI 设计规范技能，用于生成具有一致视觉风格的界面。

## 安装

将此目录放置在 `.codebuddy/skills/ui-design-guide/` 下即可使用。

## 结构

```
ui-design-guide/
├── SKILL.md              # 技能主文件（必需）
├── references/           # 参考文档
│   ├── core/            # 核心设计规范
│   ├── components/      # 组件规范
│   ├── implementation/  # 技术实现
│   ├── systems/         # 辅助系统
│   └── reference/       # 参考指南
└── assets/              # 资源文件（模板等）
```

## 快速使用

- **颜色规范**: `references/core/02-colors.md`
- **组件设计**: `references/components/`
- **快速参考**: `references/reference/04-quick-reference.md`
